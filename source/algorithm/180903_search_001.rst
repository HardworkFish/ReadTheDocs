剑指 Offer(1)
=============

## Description

::
  在一个二维数组中（每个一维数组的长度相同），每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。

相关知识点: 查找

## Solution in C++

:: 
  class Solution {
public:
    bool Find(int target, vector<vector<int> > array) {
        if(array.size()==0){
            return false;
        }
        int rows = array.size() ;
        int cols = array[0].size();
        int i = rows -1;
        int j = 0;
        while(i>=0 && j<cols){
            if(target<array[i][j])
                i--;
            else if(target>array[i][j])
                j++;
            else
                return true;               
        }
        return false;               
    }
};

 
- 运行时间：12ms
- 占用内存：1500k
